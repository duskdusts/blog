## 工具 heroku, workers
## 开始
### 部署
#### heroku
https://dashboard.heroku.com/new?template=https%3A%2F%2Fgithub.com%2Fbclswl0827%2Fv2ray-heroku
记录uuid及domain
#### workers
https://workers.cloudflare.com
创建workers
```
addEventListener(
  "fetch",event => {
     let url=new URL(event.request.url);
     url.hostname="heroku域名.herokuapp.com";
     let request=new Request(url,event.request);
     event. respondWith(
       fetch(request)
     )
  }
)
```
### 配置v2ray
地址：自选ip
端口：433
用户id：uuid
额外id：64
加密方式：auto
传输协议：ws
功能设置
伪装类型：none
伪装域名host：workers域名
path：/
底层传输安全：tls
允许不安全连接：true
