#

最近在学校广播里听到两首以前听的小众的歌，一首是【离家最近的路】，一首距离五六年的活动企划歌曲，却更加贴切了。

“不要害怕自己，长成生活的模样”

这一年很快就过去了，甚快过于以前本科繁忙的学期，我不知道是不是因为这边的作息与安排都紧密的原因。但我平常的生活却已和若干年前有很大差别，以前的我似乎有着一股充满未来的感觉去拼去玩，现在则是疲于应付，生活和工作似乎分不开了，每天的精神状态和身体健康都不太好。我很想把发生在自己身上的变化归到外部因素上，但是看着自己似乎也脱不了干系。

我发现自己还是缺些气。以前自己总觉得哈尔滨远，现在到了长沙还是觉得远，哪怕同城的朋友也很少去见一面，身体，发现自己其实还是对生活把握的底气不足，在已经打破过的规则面前依然踌躇不前。其实在面对某些期望的情感和事物面前也是如此。但其实总能发现，某些东西能够些微触发向前走的可能性，该称这些东西为什么呢，就像疲惫后外出见到的意外景象勾起多年前死去的记忆获得对某些事物的激动，或许就叫trig吧，就像示波器里trig设置好了，波形也不乱晃了，那么与我而言该怎么找好这个trig呢。

“孩子眼前的方向，是离家最近的路”

六年前，选择离家很远的哈尔滨，想要脱下束缚，却发现自己中断了很多联系，但也认清了很多东西，发现一路走来自己远比想象中更有力量，倒不必惦记着外人看着自己的模样，有决心，放手去做，都能达到。两年前，基于自己的判断和内心的选择，又选择了长沙，也让我对体制内相关的事物更了解，对自己的底气更足。

另一首是【姗姗】，看着想起了自己的模样。

“最近睡眠好吗”

入学以来睡眠一直是问题，连带的还有其他问题。记得本科床上扣手机半个点就能睡着，现在经常熬几个点，两点以前睡觉的天数都屈指可数，随之而来的是晚起、不吃早饭和工作时间的不能保证以及工作的质量下降。连带着体态的变化和饮食规律的改变，互相交织影响，任其发展，却无从改变。

“走着走着怎么脚步渐渐慢了”

对于主业的未来持怀疑态度，对于感兴趣的事物或技能没有自己的一套行之有效的流程，往往拖拖拉拉，蜻蜓点水。对很多事物都想了解与上手，但是没有专注能力和勇气，其实很多东西入门的时间也并不长，但总是在零碎时间中刷手机溜走了。

“每个人三餐一素都会有个家”

研究生是大学到社会的轻度过渡阶段，也初步了解了一些人际关系和利益关系。而自己在与学生和前辈面前交往时总保持着对人的纯粹的认知，在面对多方利益纠缠时也把握不了方向，自己是越来越不懂人了。且不说恋爱，就连周遭的社会也让人难以捉摸，多元的价值观与利益碰撞，思维已经固化，评判不了改变不了。

这是个竞掠的时代，每个人都有自己的苦楚，你我也不例外，愿经历过无数的人，此刻正经历温暖，只愿温柔的风，永远吹向春天，而你依旧在路上。
